PROBLEMATICA Y EXPLICACION DEL EJERCICIO EXTRA 

Una pequeña sala de cine necesita un sistema simple para recolectar la opinión de los clientes acerca de sus géneros favoritos
de películas y su preferencia numérica para evaluar algunos aspectos, como la calificación del servicio. El cine quiere automatizar
esta recolección para eliminar el uso de encuestas en papel, ahorrando tiempo y esfuerzo tanto del personal como de los clientes.
La solución es crear una interfaz gráfica sencilla que permita a los clientes seleccionar su género favorito de una lista y 
proporcionar una calificación entre 0 y 100. 

Explicación del Código
Este ejercicio proporciona una interfaz gráfica de usuario (GUI) en Python utilizando la librería PyQt5. El formulario permitirá 
a los usuarios seleccionar una categoría de películas y un número (que podría representar una calificación o preferencia). 
Luego, los datos seleccionados serán enviados y mostrados en la consola. La solución combina varios widgets de PyQt5 para
crear una interfaz amigable y funcional.
